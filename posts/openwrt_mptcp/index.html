WAN Bounding on OpenWRT

January 29, 2026 · 1 min · Handy Baile

Building OpenWRT with MPTCP Support#

Cloning OpenWRT#

git clone https://github.com/openwrt/openwrt.git
cd openwrt
git pull
git checkout openwrt-25.12

./scripts/feeds update -a
./scripts/feeds install -a

Configure Firmware#

make menuconfig -j$(nproc)
  • → Target System → x86
  • → Subtarget → x86_64
  • → Save → Ok

Configure Kernel#

make kernel_menuconfig -j$(nproc)
  • → Networking support → Networking options -> - MPTCP: Multipath TCP
  • → Save → Ok

Build#

make defconfig download clean world -j$(nproc)

The resulting image will be at: ./bin/targets/x86/64/openwrt-x86-64-generic-ext4-combined-efi.img.xz

Install#

The installation process is the same has the regular image.
